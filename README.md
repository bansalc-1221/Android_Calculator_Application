# Calculator App (Android)

A simple **Android Calculator Application** built using **Java in Android Studio**.
This app performs basic arithmetic operations and includes a **Splash Screen** when the app starts.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Clear button
* Simple and user-friendly UI
* Splash Screen on app launch

## Technologies Used

* **Java**
* **Android Studio**
* **XML Layout**
* **Gradle (Kotlin DSL)**

## App Screens

* Splash Screen
* Calculator Screen

## Project Structure

* **MainActivity.java** – Handles the main calculator functionality and performs arithmetic operations such as addition, subtraction, multiplication, and division.

* **SplashScreen.java** – Displays the splash screen when the application starts and redirects the user to the main calculator screen after a few seconds.

* **activity_main.xml** – Defines the user interface layout for the main calculator screen, including buttons and display area.

* **activity_splash_screen.xml** – Defines the layout design for the splash screen that appears when the application launches.

* **AndroidManifest.xml** – Contains essential information about the application, including activities, permissions, and the app's entry point.

## Purpose of the Project

The purpose of this project is to develop a simple and user-friendly **Android Calculator Application** that can perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

This project helps in understanding the fundamentals of **Android app development**, including activities, layouts, event handling, and user interface design.

The app also includes a **Splash Screen**, which enhances the user experience by displaying a welcome screen when the application launches.
